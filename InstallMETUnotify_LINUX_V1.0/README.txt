################################################
<------------DURING INSTALLATION----------------->
1-Packet manager of system will be updated.
2-Missing libraries will be downloaded.
3-Program will be installed.
Please make sure that you are online.

################################################
<------------------INSTALL-------------------->
If you have missing libraries to use program, they will be automatically downloaded.
Installing:
1-Open Terminal
2-Go to path of "InstallMETUnotify" directory. (directory which you open the README.txt)
3-Run following command
sudo chmod 777 .install.sh && ./.install.sh

Enjoy :)

################################################
<-----------CREATE ICON TO DESKTOP-------------->
If you delete your icon from desktop, don't be worry. Do followings:
1-Open Terminal
2-Go to path of "InstallMETUnotify" directory. (directory which you open the README.txt)
3-Run following command
sudo chmod 777 .createIcon.sh && ./.createIcon.sh

################################################
<--------------CHANGE PASSWORD---------------->
To change your previously saved username and password password, do followings:
1-Open Terminal
2-Go to path of "InstallMETUnotify" directory. (directory which you open the README.txt)
3-Run following command
sudo chmod 777 .uspw.sh && ./.uspw.sh
